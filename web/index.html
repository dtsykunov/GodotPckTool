Adds files into an existing .pck archive. You can add a single file or an entire ZIP at once.
  1. Drop your .pck file in the zone above
  2. Choose what to add in the zone below:
    • Single file — set the res:// destination path, e.g. res://sprites/hero.png
    • ZIP archive — all files inside are added at once; strip a leading folder to control paths, e.g. strip assets so that assets/hero.png becomes res://hero.png
  3. Click Add to PCK
